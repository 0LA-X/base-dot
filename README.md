## Yo.!
> ./setup.sh

**The dotfiles can be used by cloning the repo and running the install script**
``` bash
git clone https://github.com/0LA-X/base-dot.git

cd base-dot && ./setup.sh
```

>> [!NOTE]
> **The configs are symlinked using stow**
> **Only works on Arch based distros**
